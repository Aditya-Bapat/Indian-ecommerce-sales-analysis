# Indian E-Commerce Sales Analysis
This project analyzes sales performance across product categories, regions, and customer segments using Tableau.  
The goal is to identify patterns in revenue trends, discount strategies, and regional demand.

## Dataset
The dataset contains simulated Indian e-commerce sales data including:
- Order ID
- Order Date
- State and Zone
- Product Category
- Brand Type
- Customer Gender and Age
- Base Price and Discount Percent
- Units Sold and Revenue
- Sales Event (Festival / Normal)

## Tools Used
- Tableau (Data Visualization)
- Excel (Data Preparation)

## Key Analysis
The dashboard explores several business questions:
- Which product categories generate the highest sales?
- How do sales change over time?
- How do discounts affect revenue across categories?
- What regional differences exist in product demand?
- How do customer demographics influence sales?

## Key Insights
- Premium Lifestyle and Electronics show the highest average sales.
- Sales remain relatively stable monthly with moderate seasonal peaks.
- Higher discounts do not always result in higher revenue.
- Western and Southern regions show stronger sales performance.
- Premium brands slightly outperform mass brands in average sales.

## Dashboard Features
- Category-wise sales comparison
- Monthly sales trend analysis
- Geographic sales distribution across India
- Discount vs revenue analysis
- Customer demographic insights
